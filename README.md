## 🌑 当前版本 (v2.0 Dark) 特色说明

这个哥特式暗黑版本做了以下核心改造：

### 视觉风格
*   **强制暗黑模式**：无视系统设置。
*   **主色调**：深血红 (#8b0000)，象征血液的神秘。
*   **次要色**：暗金/铜绿，营造古老炼金术氛围。
*   **字体**：衬线字体与等宽字体混合，增强仪式感。

### 交互体验
*   **聆听之眼**：中央红色眼睛按钮，带有脉动动画。
*   **炼金计时器**：录音时显示哥特式全屏覆盖层。
*   **羊皮纸风格**：文本区域采用深色羊皮纸质感。

### 功能命名重构
*   语音转录 → **"灵魂捕获"**
*   文本生成 → **"炼金术"**
*   会议纪要 → **"仪式记录"**

### 细节处理
*   所有按钮都有血红色边框和悬停辉光效果。
*   分割线使用 ◈ 符号装饰。
*   状态指示器带有呼吸灯效果。
*   底部添加 **"MEMENTO MORI"**（记住你终将死去）的哲学铭言。
