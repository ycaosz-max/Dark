## ✨ 炼金术特性 (v2.7+)

### 1. 深度沉浸式 UI
- **哥特视觉**：采用暗红与深灰交织的视觉语言，所有组件（包括上传区、下拉框、下载按钮）均已完全汉化并哥特化。
- **仪式映射**：界面显示为“仪式记录”、“智慧秘卷”等神秘学名称，后台自动映射为精确的 AI 指令，确保生成质量。

### 2. 智能登录链路 (Vigil Auth)
- **无感通行**：优先读取部署环境的 `st.secrets`，实现打开即用的自动化体验。
- **灵魂记忆**：支持浏览器 `localStorage` 存储 API 密钥，即使在非 Secrets 环境下也能记忆您的身份。

### 3. 纯净转录逻辑
- **去噪处理**：自动解析语音转写返回的 JSON 冗余，确保“原始灵魂印记”框内只保留纯净的文字。

## 🛡️ 分支管理守则

为了维护这台古老的引擎的稳定性，请遵循以下开发仪式：

1. **`main` 分支受限**：此分支已开启保护规则，禁止强制推送（Force Push）和直接删除。
2. **实验先行**：若有新的炼金点子（Feature），请务必从 `main` 拉取新分支进行实验。
3. **归档原则**：重大阶段性成果将重命名为简洁的存档分支（如 `darkplus`）永久保留。

## 🚀 启动仪式

1. **部署环境**：建议部署于 Streamlit Cloud。
2. **配置密钥**：在 Secrets 中添加 `SILICONFLOW_API_KEY` 以获得最佳体验。
3. **开始炼金**：上传录音或实时捕获灵魂低语，选择仪式类型，见证预言书卷的诞生。
